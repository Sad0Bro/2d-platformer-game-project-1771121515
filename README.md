# 2D Platformer Game
## Description
A 2D platformer game developed using Unity and C#. The game features a side-scrolling design with various levels, enemies, and power-ups. Players control a character that can jump, run, and shoot projectiles to overcome obstacles and defeat enemies.

## Features
* Multiple levels with increasing difficulty
* Various enemies with unique behaviors
* Power-ups, such as health boosts and temporary invincibility
* Smooth character movement and jump mechanics
* Basic UI and scoring system

## Tech Stack
* Unity Game Engine
* C# programming language
* Unity physics and graphics modules

## Installation Instructions
To run the project, follow these steps:
1. Install Unity Hub and Unity Editor (version 2022.1 or later)
2. Clone the repository using `git clone https://github.com/your-username/2D-Platformer-Game.git`
3. Open the project in Unity Editor by selecting `File` > `Build Settings` > `Open Project` and navigating to the cloned repository
4. Ensure the Unity Editor is set to use the C# language and the correct .NET version

## Usage Examples
To run the game, follow these steps:
1. Open the Unity Editor and select the `2D-Platformer-Game` scene
2. Press the `Play` button in the Unity Editor to start the game
3. Use the `W`, `A`, `S`, `D` keys to control the character movement and the `Space` bar to jump
4. Use the `Mouse Left Button` to shoot projectiles

## Project Structure
The project is divided into the following folders:
* `Assets`: contains game assets, such as images, audio files, and 3D models
* `Scenes`: contains Unity scenes, including the `2D-Platformer-Game` scene
* `Scripts`: contains C# scripts, including character movement, enemy AI, and UI logic
* `Prefabs`: contains pre-built game objects, such as the player character and enemies

## Configuration
To configure the game, modify the `GameSettings` script in the `Scripts` folder. This script contains variables for game speed, difficulty, and other settings.
